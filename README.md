## Features

- Real-time text input and live preview
- Adjustable scrolling speed
- Clean, responsive interface
- Supports keyboard shortcuts for easy control

## Installation (for local development)

1. Clone the repository:

```bash
git clone https://github.com/Wunmi-hub/coding_challenge_Teleprompter.git
cd coding_challenge_Teleprompter

2. Install dependencies:
npm install

3. Run locally:
npm run dev
Open http://localhost:5173 in your browser

For production:
npm run build
